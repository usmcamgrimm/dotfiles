# dotfiles
This repository was generated by my scaffolding script.

## Scripts Usage

All scripts should be made executable first:
```bash
chmod +x *.sh
```

### `deploy.sh`
This script is designed for a Rails application using Parklife and deploys a static site to Netlify..
```bash
./deploy.sh [--dry-run]
```
- `--dry-run`: Runs the build and validation steps, but skips the actual deployment.

### `logscanner.sh`
Analyzes log files for common errors, counts the errors, and provides a summary.
```bash
./logscanner.sh <path_to_log_file>
```

### `scaffold.sh`
Automates the creation of a new Git repository.
```bash
./scaffold.sh
```
Follow the interactive prompts for repository name and branch.
